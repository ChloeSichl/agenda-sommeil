💊 Rappel Mélatonine — 21h30

21:30

Ce rappel sonne chaque soir à 21h30 pour te rappeler de prendre ta mélatonine avant de dormir.

⏰ Configurer le rappel sur iPhone

Pour recevoir une alarme chaque soir à 21h30, deux options :

Option 1 — Fichier agenda (.ics)
Ouvre le fichier rappel-melatonine-21h30.ics sur ton iPhone. Il s'importe automatiquement dans ton calendrier avec un rappel chaque soir.
Option 2 — Alarme manuelle
Ouvre l'app Horloge → onglet Alarme → appuie sur + → règle sur 21:30 → active Répéter tous les jours → nomme-la "💊 Mélatonine".
